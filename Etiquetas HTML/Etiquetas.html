En esta lectura vamos a conocer algunas de las etiquetas más usada el lenguaje de programación web, conoceremos cual es el método de implementación de cada una de ellas además de un ejemplo que a clara como debe ser la implementación de estas etiquetas a la hora de hacer o programar en un id de lenguaje de programación web.

Que es una etiqueta HTML?

Las etiquetas html se utilizan para marcar el texto de una pagina web o darle instrucciones especificas al navegador sobre como mostrar el el texto

Las etiquetas html son la que estructan y definen todo el contenido en un documento html.Cada una de estas etiquetas contienen instrucciones sencillas que indican al navegador como dar formato al texto y a definir los diversos elementos de la pagina web. Cuando aplicamos estas etiquetas a los diferentes elementos del texto, se indica al navegador como mostrarlos al usuario, lo que permite crear paginas web estructuradas y con diseno coerente.
